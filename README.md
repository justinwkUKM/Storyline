# Storyline

Turn PDFs into bold, editable visual stories with Gemini AI.

View your app in AI Studio: https://ai.studio/apps/be1463c9-c561-4b80-92af-bb38c716e0a7

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
